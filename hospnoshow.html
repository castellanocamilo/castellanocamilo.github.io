Medical Appointment No-Show Prediction

Resumen del proyecto

Este proyecto aborda el problema de la ausencia de pacientes a consultas médicas (no-show). Este fenómeno genera pérdida de recursos hospitalarios y reduce la eficiencia del sistema de salud.

El objetivo del trabajo es construir un modelo predictivo capaz de estimar la probabilidad de que un paciente no asista a su cita médica utilizando información disponible al momento de la reserva.

El dataset contiene registros históricos de citas médicas en Brasil con variables demográficas, clínicas y operativas como edad, comorbilidades, barrio y envío de SMS.

El problema se formuló como una tarea de clasificación binaria donde el modelo predice la probabilidad de no-show para permitir intervenciones operativas como confirmaciones proactivas o sobre-reservas.

Metodología

  • Limpieza y normalización de datos
  • Feature engineering (lead_time_days, age_group, variables calendario)
  • Entrenamiento de Logistic Regression y HistGradientBoosting
  • Validación cruzada estratificada

Resultados

El modelo HistGradientBoosting mostró mejor desempeño en métricas ROC-AUC y PR-AUC. El análisis de importancia de variables indicó que el tiempo entre reserva y consulta (lead_time_days) es el predictor más relevante.

El modelo puede utilizarse para priorizar intervenciones operativas como confirmaciones proactivas o re-agendamientos.
