Freelancer Income Predictor

Resumen del proyecto

Este proyecto analiza los factores que influyen en los ingresos de freelancers en plataformas de trabajo remoto como Upwork.

El objetivo del trabajo es desarrollar un modelo predictivo capaz de estimar la tarifa horaria de un profesional independiente a partir de variables como habilidades, experiencia, tipo de trabajo y características del perfil.

El modelo fue entrenado utilizando un dataset de freelancers que incluye información sobre skills, país del cliente, nivel de experiencia y categoría de trabajo.

Además del modelo predictivo, se desarrolló una pequeña aplicación interactiva que permite ingresar características de un perfil de freelancer y obtener una estimación del ingreso esperado.

Metodología

  • Análisis exploratorio de datos
  • Preprocesamiento y selección de variables
  • Entrenamiento de modelos de regresión
  • Evaluación mediante MAE, RMSE y R²
  • Construcción de interfaz interactiva para predicción

Resultados

El modelo permite estimar la tarifa horaria esperada para un freelancer en función de sus habilidades y experiencia. La aplicación interactiva facilita la visualización de predicciones en tiempo real.

Este tipo de herramientas puede ser útil para orientar decisiones profesionales dentro de plataformas de trabajo remoto.
