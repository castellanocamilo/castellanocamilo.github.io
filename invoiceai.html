Invoice Field Detection with Deep Learning

Resumen del proyecto

Este proyecto presenta un sistema de inteligencia artificial para la detección automática de campos clave en facturas. El objetivo es automatizar la extracción de información relevante como el nombre de la empresa, dirección, fecha y monto total a partir de imágenes de documentos.

El sistema utiliza técnicas de visión por computadora y aprendizaje profundo para segmentar las regiones relevantes dentro del documento. Luego se aplica OCR para extraer el texto contenido en cada campo detectado.

El dataset utilizado es ICDAR 2019 SROIE, un dataset ampliamente utilizado en investigación para extracción automática de información en recibos y facturas comerciales.

El modelo principal implementado es una arquitectura U-Net entrenada para segmentación semántica, permitiendo identificar los campos clave dentro del documento antes de aplicar reconocimiento de texto.

Metodología

  • Generación de pseudo-máscaras a partir de bounding boxes
  • Entrenamiento de modelo U-Net
  • Segmentación de campos clave
  • Extracción de texto con EasyOCR

Resultados

El modelo logró identificar correctamente los campos principales dentro de las facturas. La combinación de segmentación semántica y OCR permitió construir un pipeline completo de extracción automática de información.
